# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added
- Interactive demographic heatmap with multi-level drill-down (National → State → District → Pincode)
- Advanced analytics dashboard with trend analysis
- Regional comparison view for side-by-side analysis
- AI-powered anomaly detection using Z-score analysis
- Smart search functionality for states and districts
- IndexedDB persistence for uploaded data
- CSV export functionality
- PDF report generation with charts and statistics
- PNG export for map visualizations
- Web Workers for background CSV processing
- Date range filtering
- Geography filters (State, District, Pincode)
- Age group segmentation (0-5, 5-17, 17+)
- Responsive design for mobile and desktop
- GitHub Pages deployment with automated CI/CD
- Comprehensive user guide with tutorial images
- Developer documentation and contributing guidelines

### Technical Stack
- React 18 with TypeScript
- Vite build tool
- Tailwind CSS for styling
- Zustand for state management
- Leaflet for interactive maps
- Recharts for data visualization
- IndexedDB (idb) for client-side storage

- README.md with comprehensive project overview

- CONTRIBUTING.md with development guidelines
- CODE_OF_CONDUCT.md for community standards
